## <br /> <a name="introduction">🎨 Introduction</a>

Plongez dans une immersion visuelle inspirée de la Chine contemporaine, grâce à un slider parallaxe synchronisé combinant effet de profondeur, flou artistique et transitions élégantes.

Un rendu cinématographique captivant dès le premier scroll, pensé pour éveiller les sens et l’imaginaire.

Un projet idéal pour les portfolios visuels, sites culturels ou expériences web à forte dimension artistique et narrative.

## <br /> <a name="technologies">🛠️ Technologies</a>

- HTML5 sémantique et accessible
- CSS3 moderne avec variables, transitions et media queries
- JavaScript ES6 clair et modulaire
- [Swiper.js](https://swiperjs.com/) pour le slider interactif
- Effet parallax natif de Swiper

## <br /> <a name="fonctionnalités">🎯 Fonctionnalités</a>

- Slider principal et fond synchronisés avec effet parallaxe
- Mise en page responsive desktop et mobile
- Titre et description dynamiques à la première slide
- Lightbox visuelle avec effet de zoom au clic
- Transitions fluides entre les slides
- Blurs en arrière-plan pour une ambiance atmosphérique
- Affichage adaptatif selon la taille d’écran
- Navigation intuitive via le scroll ou la souris

## <br /> <a name="installation">🚀 Installation</a>

### ✅ Prérequis

- [Google Chrome](https://www.google.com/) &nbsp;—&nbsp; Navigateur moderne
- [Visual Studio Code](https://code.visualstudio.com/) &nbsp;—&nbsp; Éditeur de code
- [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) &nbsp;—&nbsp; Extension VS Code

### 📥 Cloner le projet

```bash
git clone https://github.com/ValentinMadiot/background-slider-parallax-china_js
cd background-slider-parallax-china_js
```

### ▶️ Lancer le projet

Il suffit d’ouvrir le fichier `index.html` dans un navigateur, ou d’utiliser l’extension **Live Server** sur VS Code pour un aperçu dynamique.
